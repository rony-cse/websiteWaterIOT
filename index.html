Prevent the clot in Sewrage line to avoid unexpected flood.

  • A dependable survey report from long-time real-time data
  • Basic unwanted waste removal system to avoid clot.
  • Future overflow of sewage drain could be avoided

This is an IoT base system which would collect water levels of specific points multiple times every hour and store that data in a cloud base database via wifi. Our Web app will receive the data and process it to determine the use load of that line and show it in graphs for minute, hour, day, month or even seasonal basis. Also there will be an alarm system if at any point the drain overflowed. To remove the plastic and other larger size waste from the drain , a conveyor system will automatically grab and pull the waste out. In case of automation failure of jammed manual override request will be send to control center immediately for maintenance or human base deep cleaning.
